Description

La source de données OPC permet de lire les données d'un serveur OPC

Configuration

Toutes les sources de données nécessitent un Nom, qui peut avoir n'importe quelle description.

Le Hôte indique l'adresse locale du client.

Le Domaine indique le domaine où se trouve le client. PS : Le client doit être nécessairement dans un domaine pour fonctionner avec OPC.

Le User indique le nom de l'utilisateur à partir du système d'exploitation - OS.

Le Mot de passe indique le mot de passe du système d'exploitation.

Le Serveur indique quels sont les serveurs disponibles pour les données fournies.

Le Période de mise à jour indique la période de mise à jour à partir des données de la source de données.

Le Mode de création indique comment les balises OPC seront créées sur le client. Nous avons deux options : Parcourir les Tags - où tous les Tags disponibles dans le Serveur seront affichés et Ajouter des Tags - où sera ajouté un Tag disponible du serveur
