Vue d'ensemble

Un moteur de rendu graphique est utilisé pour fournir l'état historique global d'un point sur une certaine période de temps ou un certain nombre de valeurs. Les informations agrégées qui peuvent être fournies dépendent du type de données des valeurs à rendre. Ainsi, seuls les types de moteur de rendu qui s'appliquent au type de données du point actuel sont répertoriés dans la sélection Type.

Note! Les rendus de graphiques dépendent de données enregistrées. Si votre point a un type de journalisation "Ne pas journaliser", votre outil de rendu de graphiques peut ne pas s'afficher comme prévu. L'exception potentielle est le graphique Table, qui peut utiliser des valeurs de point mises en cache dans la mémoire, mais seulement si ces valeurs sont disponibles. Le redémarrage du système efface tous les caches de valeurs ponctuelles.

Tableaux graphiques

Ce moteur de rendu affiche un nombre donné (le Limite) de valeurs historiques les plus récentes pour le point. Les valeurs sont affichées dans un tableau fournissant la valeur (rendue par le texte), la valeur temps, et toute annotation applicable. Ce tableau est disponible pour tous les types de données.

Graphique Statistiques

Ce graphique affiche un tableau d'informations statistiques calculées à partir des valeurs ponctuelles historiques pour la période donnée Période temporelle. Les données réelles affichées dépendent du type de données du point. Ce graphique est disponible pour tous les types de données.

Binaire et Multistate : Ces types de données produisent une liste de statistiques "start and runtime" pour chaque état que le point a connu pendant la période de temps. Chaque élément "start and runtime" fournit le nombre de fois que l'état a été saisi ("starts"), la durée totale pendant laquelle le point était dans l'état ("runtime"), et le pourcentage de la durée totale de la période que représente le runtime.

Numérique : Les points numériques produisent les valeurs statistiques suivantes pour la période de temps : valeur minimale et moment où elle s'est produite, valeur maximale et moment où elle s'est produite, moyenne et somme (adaptée au comptage des impulsions).

Alphanumérique : Le nombre de fois que le point a changé au cours de la période est fourni.

Graphiques d'images

Ce graphique affiche un diagramme à barres linéaires des valeurs des points sur la période donnée. Il est disponible pour les points avec des types de données numériques ou multi-états.
