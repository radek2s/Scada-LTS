La source de données virtuelle est une source de données sans aucune connexion à un système externe ou physique. Elle existe entièrement dans l'espace mémoire du système. Les données qu'elle produit sont soit aléatoires, soit déterministes, selon la configuration des points.

Les sources de données virtuelles peuvent être utiles de plusieurs façons:

  • Un moyen simple de valider que votre système fonctionne
  • Il peut créer des données prévisibles avec lesquelles vous pouvez comparer vos données réelles
  • Vous pouvez contrôler les équipements de manière plus complexe. Par exemple, si l'éclairage de votre maison est contrôlé par le système, vous pouvez utiliser un point binaire aléatoire pour éteindre et allumer les lumières pendant que vous êtes en vacances, en donnant l'impression que vous êtes toujours chez vous.
  • Tout endroit où des données déterministes ou aléatoires sont utiles

La configuration de la source de données est simple. Elle ne nécessite qu'un nom, et la période de temps entre chaque mise à jour des données. Par exemple, si vous choisissez "2 minutes", alors toutes les deux minutes, les algorithmes qui modifient les valeurs des points virtuels pour cette source de données seront exécutés.
