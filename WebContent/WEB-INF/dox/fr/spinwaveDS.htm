Vue d'ensemble

La source de données Spinwave est utilisée pour recueillir des données à partir de dispositifs Spinwave connectés à une passerelle locale, et accessibles via une communication RS232 ou RS485 (nécessite un adaptateur). Cette source de données reçoit des mises à jour régulières des dispositifs connectés ; elle ne pollue pas.

Configuration

Chaque source de données nécessite un Nom, qui peut être n'importe quelle description. La sélection Port fournit une liste de tous les ports série disponibles sur le matériel hôte. Cela correspond au port dans lequel la passerelle Spinwave est connectée. La version Message dépend du matériel que vous utilisez. Consultez la documentation de votre Spinwave pour plus de détails.

Les adresses des capteurs sont généralement trouvées en utilisant la fonctionnalité d'écoute des capteurs décrite ci-dessous, bien que si vous connaissez les adresses uniques de votre matériel, vous pouvez les saisir manuellement. Après avoir saisi la valeur de l'adresse, cliquez sur l'icône pour l'ajouter à la liste des adresses connues.

Capteur d'ondes spongieuses à l'écoute

Une fois que la passerelle Spinwave a été connectée à l'hôte, il est possible de lancer un auditeur qui attend l'arrivée des messages. En général, les appareils sont configurés pour envoyer des messages toutes les 21 secondes, mais d'autres réglages sont possibles. Pour ajouter les adresses de tous les dispositifs qui ont été écoutés à la liste des adresses de capteurs, cliquez sur le bouton "Ajouter les capteurs trouvés à la liste".
