Vue d'ensemble

La source de données SNMP est utilisée pour recueillir les données des appareils qui mettent en œuvre le protocole SNMP. Cette source de données effectue des interrogations et accepte également les traps SNMP.

Configuration

Chaque source de données nécessite un Nom, qui peut être n'importe quelle description. La période Mise à jour détermine à quelle fréquence les points SNMP seront interrogés pour obtenir des données. Les autres champs sont nécessaires pour configurer les détails de la communication SNMP.

Test de l'hôte du PNMS

Le test de l'hôte peut être utilisé pour tester à la fois la configuration de la source de données et l'existence et la valeur d'un OID (Object IDentifier) donné. Entrez l'OID à tester et cliquez sur "Get OID value". Les paramètres de configuration actuels seront utilisés pour envoyer une demande d'obtention SNMP. En cas de succès, la valeur de l'OID sera affichée. Sinon, un message décrivant l'erreur sera fourni.
