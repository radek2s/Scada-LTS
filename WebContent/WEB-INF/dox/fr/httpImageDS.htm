Vue d'ensemble

Cette source de données est utilisée pour collecter des images de sites web disponibles sur Internet, ou d'équipements internes disponibles sur un intranet. À l'aide de mécanismes similaires à ceux d'un navigateur web, la source de données demande des informations à des appareils compatibles HTTP, puis enregistre les images dans une zone protégée du système de fichiers. Il s'agit d'une source de données de sondage.

Configuration

Chaque source de données nécessite un Nom, qui peut être n'importe quelle description. Le Période de mise à jour détermine la fréquence de la requête HTTP.
