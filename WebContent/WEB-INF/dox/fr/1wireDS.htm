Vue d'ensemble

La source de données 1-wire est utilisée pour recueillir des données à partir d'un réseau local 1-wire, accessible via une communication RS232 ou RS485 (nécessite un adaptateur). Il s'agit d'une source de données d'interrogation.

La source de données est capable de tolérer des reconfigurations physiques du réseau. Lorsque des modifications de la topographie d'un réseau sont détectées, un nouveau balayage automatique est effectué. Si tous les points sont toujours localisés après le balayage, aucun événement n'est soulevé.

Configuration

Chaque source de données nécessite un Nom, qui peut être n'importe quelle description. La sélection Port fournit une liste de tous les ports série disponibles sur le matériel hôte. Cela correspond au port dans lequel le 1-wired est connecté. La sélection Période de mise à jour détermine la fréquence à laquelle le réseau est interrogé pour des données.

Scan du réseau

Les réseaux peuvent être scannés pour détecter les appareils à l'aide de l'utilitaire de scan. Cette fonctionnalité permet de scanner les chemins connectés par des coupleurs pour localiser tous les équipements du réseau. Les résultats sont affichés dans un tableau fournissant l'adresse 1-wire et une description de l'appareil. Pour ajouter rapidement un appareil comme point, cliquez sur l'icône associée. Un nouveau point sera créé avec les attributs du périphérique par défaut.
