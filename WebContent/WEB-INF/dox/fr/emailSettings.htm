Les paramètres e-mail sont utilisés pour tous les envois par mail de Scada-LTS.

L'hôte SMTP est le nom d'hôte du serveur sur lequel le MTA (Mail Transfert Agent) est exécuté. Si le serveur requiert une authentification, cochez la case autorisation d'utilisation, et remplissez les champs nom d'utilisateur et Mot de passe. Cliquez la case Activer TLS si le serveur l'exige.

Le champ d' adresse est l'adresse d'envoi utilisée pour tous les mails. Le champ de nom est un champ facultatif pour spécifier un nom nom propre ou un alias pour désigner l'expéditeur.

Utiliser la boite de sélection type de contenu que vous souhaitez voir dans les mails. Par défaut HTML et texte est sélectionné. Note: L'envoi par mail de rapports planifiés est toujours HTML-seulement à cause du contenu de ces mails.

Une fois les paramètres définis vous pouvez tester vos règlages en cliquant sur l'icône . Ceci tentera d'envoyer un mail avec vos nouveaux paramètres.
