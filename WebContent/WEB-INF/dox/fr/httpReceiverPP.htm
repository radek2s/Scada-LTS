Le paramètre HTTP nom est la clé utilisée pour livrer les données à la source de données du récepteur HTTP. Elle est fournie sous la forme de la partie "pointName" dans les valeurs de point formatées soit comme "pointName=pointValue" ou __pointFoo=pointName&__valueFoo=pointValue'. Vous pouvez utiliser la fonctionnalité d'écoute du récepteur HTTP dans la configuration de la source de données du récepteur pour déterminer les noms des paramètres disponibles dans vos demandes.

Le Type de données est tout type de données valide. Voir l'aperçu général du système pour plus d'informations sur les types de données.

Si le type de données est Binaire, une valeur Binaire 0 doit être fournie. Si la valeur du point dans la demande correspond à cette valeur, la valeur attribuée au point sera 0 ; sinon, elle sera 1.
