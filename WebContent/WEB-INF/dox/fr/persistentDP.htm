À FAIRE Le Type de données est la représentation de la valeur par Mango. Les données reçues sont automatiquement converties dans le type de données requis.

Le Feed ID et Data stream ID sont utilisés pour identifier de manière unique un flux de données. Les demandes de plusieurs flux au sein d'un même flux sont consolidées pour des raisons d'efficacité.

Pour le type de données binaires, la valeur Binaire 0 peut être définie. Si la valeur trouvée dans le contenu correspond, le point sera mis à jour avec 0. Sinon, le point est mis à jour avec 1.

Pour vous assurer que votre flux de données existe, cliquez sur l'icône associée.
