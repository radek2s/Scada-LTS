Vue d'ensembles

La plupart des types de points dans une source de données Galil reflètent les valeurs dans l'équipement Galil. L'exception est le type de point de commande, qui est plus abstrait. Les détails sont fournis ci-dessous.

Tous les types de points nécessitent un nom pour l'affichage, qui peut être n'importe quelle valeur.

Commandes

Le type de point de commande permet d'envoyer des commandes arbitraires à l'équipement Galil. Les commandes peuvent être entrées dans la liste de surveillance, à partir de vues graphiques ou via des gestionnaires d'événements de points de consigne. La valeur de la commande et la valeur de la réponse sera enregistrée dans l'historique des points (l'un après l'autre), sauf si une erreur se produit, auquel cas la commande sera enregistrée et un événement décrivant l'erreur sera déclenché.

Entrée

Les types de points d'entrée utilisent les commandes @IN et @AN pour récupérer les valeurs actuelles. Ces points ne sont pas paramétrables. La commande Type de donnéesdétermine quelle commande sera utilisée pour demander la valeur. Le Numéro d'entrée est l'index de la valeur à récupérer. Pour les points numériques, une mise à l'échelle linéaire de la valeur peut également être effectuée automatiquement en définissant les valeurs Echelle en conséquence.

Sortie

Les types de points de sortie utilisent les commandes CB, SB et @OUT pour récupérer et définir des valeurs de sortie binaires. Le numéro de sortie Output est l'indice de la valeur à gérer.

Indiquer la position

Les types de points de position Tell utilisent la commande TP pour récupérer la position d'un axe particulier. Ces points ne sont pas paramétrables. Le Axe est le label de l'axe requis. La mise à l'échelle des valeurs linéaires peut être effectuée automatiquement en définissant les valeurs Echelle en conséquence. Sélectionnez Round to integer pour arrondir les valeurs décimales de la valeur de récupération (et de mise à l'échelle).

Variable

Les types de points variables utilisent la commande "Var=" pour définir et récupérer les valeurs des variables dans l'équipement. La commande Nom de la variable est le nom de la variable souhaitée. Les valeurs à définir et à recevoir sont automatiquement converties en fonction du type de données donné Data type.
