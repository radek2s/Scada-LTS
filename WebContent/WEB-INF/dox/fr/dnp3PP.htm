Configuration

Vous pouvez ajouter un seul point, définissant son index, ou Ajouter plusieurs points de données définissant les index minimum et maximum.

Commandes

Pour les points de type OUTPUT, vous devez définir les paramètres de commande à utiliser par défaut, s'ils ne sont pas spécifiés pendant l'exécution. Pour les points de type SORTIE BINAIRE, les commandes valides (qui doivent être exécutées sur l'icône de liste de surveillance appropriée, ainsi que sur d'autres sources de données) sont:

pon/poff -> impulsion on/off

lon/loff -> verrouillage on/off

fermer/voyager -> fermer/voyager

Toutes les commandes peuvent être suivies de deux paramètres, la commande TIME ON et TIME OFF en millisecondes. Par exemple : pon,500,200

Il est possible de ne saisir que "1", "0", "true" ou "false". Dans ce cas, la commande envoyée sera la commande par défaut spécifiée au point.

Exemple

Point binaire enregistré comme FERMETURE/DECOUPE, TEMPS ON 1000 e TEMPS OFF 0

1 -> le système enverra la commande CLOSE avec TIME ON 1000 et TIME OFF 0

0 -> le système enverra la commande TRIP avec TIME ON 1000 et TIME OFF 0

pon -> le système enverra la commande PULSE ON avec TIME ON 1000 et TIME OFF 0

poff,300,0 -> le système enverra la commande PULSE OFF avec TIME ON 300 et TIME OFF 0
