Vue d'ensemble

Une référence à un point JMX consiste en un Nom d'objet, un Nom d'attribut, et éventuellement - si l'attribut est un type de données composite - un Nom d'élément composite. La façon la plus simple de trouver des valeurs est d'utiliser la fonction d'inspection du serveur JMX.

Le point Type de données est la représentation Scada-LTS de la valeur, et non le type de valeur JMX. Quoi qu'il en soit, il est préférable de faire correspondre le type de données aussi étroitement que le type JMX pour obtenir les meilleurs résultats.
