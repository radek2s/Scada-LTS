Le Type de données est la représentation de la valeur par Mango. Les données reçues sont automatiquement converties dans le type de données requis. Le Valeur RegEx détermine où dans le contenu renvoyé trouver la valeur du point. Voir la documentation "Expressions régulières" pour plus d'informations. Notez que seule la première instance de l'expression trouvée dans le contenu est utilisée.

La case à cocher Ignorer en cas de disparition indique si les valeurs manquantes sur une page doivent être considérées comme des conditions d'erreur. Si elle n'est pas cochée et que la valeur n'est pas trouvée dans le contenu, un événement d'exception de source de données sera déclenché. Si elle est cochée, l'événement est supprimé, et le point n'est pas mis à jour.

Pour le type de données binaires, la valeur Binaire 0 peut être définie. Si la valeur trouvée dans le contenu correspond, le point sera mis à jour avec 0. Sinon, le point est mis à jour avec 1. Pour le type de données numériques, un Nombre format peut être défini, ce qui fournit des informations sur la façon dont la valeur du texte trouvée par la RegEx doit être analysée. Voir la documentation "Formats des nombres" pour plus d'informations.

A Temps RegEx peut être défini pour localiser l'horodatage de la valeur dans le contenu renvoyé. Si le RegEx est défini, un Format de l'heure doit également être saisi, ce qui fournit des informations sur la manière dont l'heure du texte trouvée par le RegEx doit être analysée. Voir la documentation "Formats de date/heure" pour plus d'informations.

Pour tester votre RegEx par rapport au contenu actuel à l'URL donnée, cliquez sur l'icône associée.
