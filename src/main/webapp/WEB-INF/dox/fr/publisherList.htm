La liste liste des éditeurs répertorie tous les éditeurs définis dans le système, en affichant le nom, le type, la description de la configuration et le statut de chacun. Configuration dépend du type d'éditeur. Le Statut détermine si l'éditeur est activé () ou désactivé (). En cliquant sur l'icône d'état, on fait basculer l'état de l'éditeur.

Pour ajouter un nouvel éditeur, sélectionnez le type d'éditeur dans la liste et cliquez sur l'icône en haut de la liste. Pour modifier un éditeur existant, cliquez sur l'icône associée à l'éditeur. Pour supprimer un éditeur, cliquez sur l'icône associée à l'éditeur.
