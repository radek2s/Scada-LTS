Vue d'ensembles

La source de données Galil est utilisée pour échanger des données avec l'équipement Galil DMC-21x2 accessible via un réseau I/P. L'équipement peut se trouver dans un réseau local ou un intranet, ou peut également se trouver n'importe où sur Internet. Il s'agit d'une source de données de sondage.

Configuration

Chaque source de données nécessite un Nom, qui peut être n'importe quelle description. Les paramètres Hôte et Port déterminent comment trouver l'équipement sur le réseau. L'hôte peut être un nom de domaine ou une adresse IP. Les champs Timeout et Retries déterminent le comportement du système en cas d'échec de l'interrogation. La source de données attend le nombre de millisecondes de délai d'attente donné pour une réponse de l'équipement. Si elle n'est pas reçue, la demande sera réessayée le nombre de fois indiqué. La période Mise à jour détermine la fréquence à laquelle l'équipement est interrogé pour des données.

Une fois qu'une connexion est établie avec l'équipement, elle est maintenue en vie aussi longtemps que l'équipement le permet. Les connexions fermées provoquent une tentative de réouverture automatique.

Test de commande

La connectivité et l'état des équipements peuvent être testés en utilisant la section Test de la commande Galil. Cette fonction ouvre une nouvelle connexion sur laquelle envoyer la commande, de sorte qu'elle n'entre en conflit avec aucune source de données en cours d'exécution. Toutefois, si l'équipement n'autorise qu'un nombre limité de connexions simultanées, la connectivité pourrait échouer.

La valeur Commande peut être toute chaîne de commande valide (à l'exclusion du terminateur CRLF, et éventuellement à l'exclusion d'un point-virgule fermant). Le résultat de la commande sera affiché sous le bouton Exécuter. Si la connexion à l'équipement a échoué, le message d'erreur sera affiché. Si aucune réponse n'est reçue, la connexion s'interrompra après la durée de temporisation donnée. Si une réponse d'erreur est reçue (c'est-à-dire si '?' est reçu), le message "(réponse d'erreur)" sera affiché.
