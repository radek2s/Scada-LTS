Cette page affiche les commandes de modification d'un point. L'icône est un lien vers la page, en lecture seule, des détails du point; elle n'est visible que pour les points enregistrés.

Le nom du point est obligatoire, et peut contenir n'importe quel texte. Ce nom définira visuellement ce point pour l'ensemble du système.

Le nom de la source données est affichée. L'icône vous redirigera directement vers page d'édition de la source de donnée qui contient ce point.
