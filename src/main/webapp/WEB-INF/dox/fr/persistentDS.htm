Vue d'ensemble

À faire Cette source de données est utilisée pour collecter des informations génériques de Pachube. Les données sont organisées sur Pachube en flux, qui sont des collections de flux de données. Puisqu'un flux de données correspond à un point Mango, la définition d'un point Pachube est faite en spécifiant à la fois un flux et un identifiant de flux. Il s'agit d'une source de données de sondage.

Configuration

Chaque source de données nécessite un Nom, qui peut être n'importe quelle description. Le Période de mise à jour détermine la fréquence de la requête HTTP vers Pachube. Notez que - en fonction de votre compte chez Pachube - des restrictions de requête API peuvent s'appliquer. Voir http://community.pachube.com/api#rate_limitpour plus de détails.

La touche API est nécessaire pour accéder aux données privées. Vous pouvez la trouver dans la zone "mes paramètres" lorsque vous vous connectez à votre compte Pachube. Les champs Temps mort et Réessaye déterminent combien de temps il faut attendre que le serveur web renvoie les données, et combien de fois il faut réessayer la connexion en cas de défaillance. Si toutes les tentatives sont épuisées, un événement d'exception de source de données est déclenché.
