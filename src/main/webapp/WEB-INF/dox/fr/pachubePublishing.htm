Vue d'ensemble

L'éditeur Pachube envoie chaque mise à jour de point (ou changement, selon le cas) à Pachube en utilisant la clé API, l'ID de flux et l'ID de flux de données donnés comme une requête HTTP distincte.

Propriétés

La clé API Key est nécessaire pour mettre à jour les données privées. Vous pouvez la trouver dans la zone "mes paramètres" lorsque vous vous connectez à votre compte Pachube. Les paramètres Temps mort et Réessayer sont utilisés pour gérer les pannes.

Réponse du HTTP

Tout autre code de réponse que "200 OK" est considéré comme une erreur, et une alarme sera déclenchée. Notez que - en fonction de votre compte chez Pachube - des restrictions de demande d'API peuvent s'appliquer. Voir http://community.pachube.com/api#rate_limit pour plus de détails.

Points publiés

La section Points énumère les points à publier. La publication des points peut être activée () ou désactivée (). Désactiver un point permet de ne pas l'utiliser sans avoir à le supprimer explicitement. Les valeurs ID de l'aliment et ID du flux de données déterminent le flux de données qui sera mis à jour.

Pour ajouter un point à la liste de points, sélectionnez-le dans la liste de points et cliquez sur l'icône . Pour supprimer un point existant, cliquez sur l'icône qui lui est associée.
