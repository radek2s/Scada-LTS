Vue d'ensemble

La source de données JMX est utilisée pour surveiller les valeurs exposées par un serveur JMX, qu'il soit local à l'instance ou distant. Les valeurs disponibles varient considérablement en fonction des applications exécutées dans l'instance JVM, mais il existe toujours des valeurs communes, comme celles associées à la mémoire JVM.

Configuration

Chaque source de données nécessite un Nom, qui peut être n'importe quelle description. La case à cocher Utiliser le serveur JMX local indique si le serveur JMX de la JVM locale doit être utilisé. Sinon, la case Remote JMX server address peut être spécifiée.

Le période de mise à jour détermine la fréquence à laquelle le serveur JMX est interrogé pour des données. La vérification Quantifier entraîne un retard dans le démarrage de la source de données, de sorte que les sondages se produisent à un moment "arrondi" (par exemple, si le sondage est effectué toutes les 10 secondes, les sondages se produiront à la 10e seconde, à la 20e seconde, etc.). Sinon, le scrutin commencera immédiatement.
