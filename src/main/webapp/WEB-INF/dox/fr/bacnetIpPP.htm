Le Dispositif à distance IP est l'adresse IP à laquelle le dispositif peut être trouvé. Le Port du dispositif à distance est le numéro de port auquel il écoute les demandes.

Le Type d'objet est le type d'objet auquel des valeurs seront demandées. Le Numéro d'instance de l'objet est l'identifiant unique de l'objet/type dont les valeurs seront demandées. Les points demandent la Valeure présente d'une instance d'objet.

Le Utiliser l'abonnement COV détermine si Mango va tenter de s'abonner aux événements COV pour l'objet sur l'appareil. Tous les objets ne prennent pas en charge les abonnements COV, et tous les appareils ne les prennent pas en charge, même pour les objets qui le feraient normalement.

Utiliser Settable pour déterminer si l'interface permettra aux utilisateurs d'entrer la valeur avec laquelle mettre à jour l'objet. Seules les valeurs de "sortie" sont considérées comme étant paramétrables.

Le champ Type de données détermine le type de données Mango que le point aura. Les valeurs renvoyées par le réseau BACnet seront automatiquement converties dans le type requis.
