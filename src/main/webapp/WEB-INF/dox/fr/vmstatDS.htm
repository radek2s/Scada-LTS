Vue d'ensemble

La source de données VMStat est utilisée pour surveiller divers aspects des performances d'un hôte *nix. Lorsque la source de données démarre, un processus est lancé qui exécute le programme vmstat, et la sortie de ce processus est surveillée.

Configuration

Chaque source de données nécessite un Nom, qui peut être n'importe quelle description. Le paramètre Les secondes du sondage détermine la fréquence à laquelle le processus vmstat rapportera les données à Scada-LTS. Le paramètre Échelle de sortie peut être réglé pour ajuster automatiquement les valeurs d'octets à différentes échelles, y compris les octets, k, KB, m et MB.
