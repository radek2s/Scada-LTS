Vue d'ensemble

Cette source de données est utilisée pour collecter des informations génériques à partir des courriels reçus dans une boîte aux lettres POP3 donnée. Les messages dans la boîte aux lettres peuvent avoir été reçus de n'importe quelle source. En utilisant des mécanismes similaires à ceux d'un client de messagerie, la source de données se connecte à la boîte aux lettres POP3 et traite individuellement les messages qui s'y trouvent. Les messages sont supprimés après le traitement. Des expressions régulières sont utilisées pour extraire les informations souhaitées. Il s'agit d'une source de données de sondage.

Configuration

Chaque source de données nécessite un Nom, qui peut être n'importe quelle description. Le période de contrôle de la boîte aux lettres détermine la fréquence à laquelle la boîte aux lettres est contrôlée pour les messages. Le serveur POP3 est le nom de domaine ou l'adresse IP du serveur qui héberge la boîte aux lettres. Le port POP3 standard 110 est utilisé pour la connexion. Le nom d'utilisateur de la boîte aux lettres et le mot de passe de la boîte aux lettres doivent être fournis pour l'accès.

La zone Contenu du test est l'endroit où un échantillon de contenu de courriel peut être mis à l'essai pour tester le RegEx d'un point. Pour plus d'informations, voir la documentation sur les "propriétés des points POP3".
