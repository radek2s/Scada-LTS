Vue d'ensemble

La source de données de l'auditeur NMEA est utilisée pour recevoir des messages provenant d'équipements conformes à la norme NMEA 0183. Les appareils sont connectés via les ports RS232 ou RS485 (nécessite un adaptateur). Cette source de données reçoit des mises à jour régulières des appareils connectés ; elle ne pollue pas.

Configuration

Chaque source de données nécessite un Nom, qui peut être n'importe quelle description. La sélection Port fournit une liste de tous les ports série disponibles sur le matériel hôte. Cela correspond au port dans lequel l'équipement NMEA est connecté. La sélection Baud rate est la vitesse à laquelle votre équipement peut communiquer ses messages. En général, cette valeur est de 4800, mais d'autres valeurs sont possibles. Consultez votre documentation NMEA pour plus de détails. Les valeurs de checksum sont validées si elles sont incluses dans le message. Si la validation de la somme de contrôle échoue, le message est ignoré.

Ecouteur du message NMEA

Une fois que l'équipement NMEA a été connecté à l'hôte, il est possible de démarrer un auditeur qui attend l'arrivée des messages. Le taux d'arrivée des messages dépend de l'équipement. Les messages seront affichés en excluant le message "enveloppe", c'est-à-dire l'indicateur de début "$", l'indicateur de fin CRLF, et la valeur "*" et la somme de contrôle si elle est présente.

Les messages affichés peuvent être copiés et collés dans la case Message de test de la regex pour être testés par rapport à la regex de vos points.
