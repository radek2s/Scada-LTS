Vue d'ensembles

Une vue graphique est similaire à une liste de surveillance, mais fournit à la place une représentation graphique des points sélectionnés. Les valeurs des points et les temps de valeur se mettent à jour automatiquement - vous n'avez pas besoin de rafraîchir la page. (En fait, le système fonctionne plus efficacement si vous ne le faites pas).

Sélection d'une vue graphique

Un utilisateur peut avoir plusieurs vues graphiques, mais ne peut en visualiser qu'une à la fois. Utilisez le menu déroulant pour sélectionner la vue que vous souhaitez afficher. Pour créer une nouvelle vue, cliquez sur l'icône d'ajout () dans le coin supérieur droit de la section de la liste de surveillance. Pour modifier une vue existante, cliquez sur l'icône d'édition (). Dans les deux cas, la page Vue graphique edit s'affichera.

Vous pouvez voir dans votre liste des vues graphiques que vous n'avez pas créées. Si c'est le cas, ces vues ont été "partagées" avec vous par la personne qui les a créées. Vous serez au moins autorisé à les visualiser, mais vous ne pourrez peut-être pas définir de valeurs via la vue si vous n'avez pas reçu l'autorisation de les "définir". Si vous ne souhaitez plus avoir la vue dans votre liste, utilisez l'icône de suppression () pour "défaire" la vue. (Le dépannage ne supprime pas la vue).

Images de fond

Les vues graphiques peuvent éventuellement être présentées sur des images de fond. Les arrière-plans peuvent être définis dans la page d'édition des vues graphiques.

Points dans les vues graphiques

Un point peut appartenir à plusieurs vues graphiques, ce qui vous permet de créer des vues aussi spécifiques que vous le souhaitez.

La représentation de chaque point de votre point de vue dépend de la configuration de ce point. Les points peuvent être rendus comme une simple valeur courante, ou un ensemble d'icônes arbitraires peut être utilisé pour représenter visuellement la valeur du point. Lorsque l'utilisateur passe sur le point avec la souris, un ensemble de contrôles s'affiche (sauf s'ils ont été explicitement supprimés). Les contrôles réellement affichés dépendent également de la configuration du point. En passant la souris sur l'icône info (), la valeur actuelle du point et la dernière modification seront affichées. Les points qui sont paramétrables (et pour lesquels vous avez la permission de le faire) afficheront une icône de paramétrage (). Si vous passez votre souris sur cette icône, vous verrez des options pour définir le point. Les points qui sont configurés pour l'établissement de graphiques afficheront une icône de graphique (). Si vous passez la souris sur cette icône, le graphique actuel s'affiche. Les graphiques sont mis à jour automatiquement, comme les valeurs et les heures, de sorte que lorsque de nouvelles données sont disponibles, le graphique se mettre à jour. Si une icône d'avertissement () est affichée à côté d'un point, elle indique que celui-ci est inactif, soit parce que lui ou sa source de données sont désactivés. Un indicateur d'attente () signifie que le point n'a actuellement aucune valeur, et qu'il attend probablement que sa valeur soit reçue.

Plein écran

En mode plein écran, le contenu de la représentation graphique qui sera affichée à l'écran ne sera que les éléments présents dans la représentation. C'est-à-dire que le reste, comme l'en-tête, ne sera pas affiché.

Pour quitter le mode plein écran, utilisez le raccourci suivant : "Ctrl + Shift + F".
