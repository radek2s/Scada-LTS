Ce panneau permet la modification de divers attributs d'affichage pour le point de vue. S'il est saisi, la valeur Remplacement du nom du point sera affichée dans le popup d'information du point au lieu du nom donné au point. S'il n'est pas saisi, le prénom est affiché. La valeur Settable override permet de faire apparaître un point paramétrable comme non paramétrable dans le contexte de la vue.

La couleur Background color remplace la couleur par défaut de l'interface. Si elle est laissée vide, la couleur par défaut est utilisée. Entrez "transparent" pour ne pas avoir de couleur de fond ou de bordure affichée (utile si l'image de fond de la vue fournit déjà un contexte d'arrière-plan pour le point de vue). Sinon, toute valeur acceptable comme couleur CSS peut être saisie. Seule la couleur valeur doit être fournie. Par exemple, les valeurs valides pour le rouge comprennent (à l'exclusion des guillemets) "#FF0000", "red", "#F00", "rgb(255, 0, 0), et "rgb(100%, 0%, 0%)". Une description simple des valeurs de couleur valides peut être trouvée ici. Pour des détails exhaustifs, voir le W3C

La case à cocher Display controls détermine si les contrôles de survol seront affichés pour un point de vue. La suppression des contrôles de survol des points peut réduire l'encombrement dans les vues denses. Cela peut également donner une vue plus "statique", ce qui peut être utile pour les vues à distance.
