Vue d'ensemble

Cette source de données est utilisée pour recueillir des informations génériques à partir des sites web disponibles sur l'internet, ou des équipements internes disponibles sur un intranet. En utilisant des mécanismes similaires à ceux d'un navigateur web, la source de données demande des informations à des appareils compatibles HTTP, puis utilise des expressions régulières pour extraire les informations souhaitées. Il s'agit d'une source de données de sondage.

Configuration

Chaque source de données nécessite un Nom, qui peut être n'importe quelle description. Le Période de mise à jour détermine la fréquence de la requête HTTP. Le URL est l'adresse de la page qui doit être analysée. Utilisez l'icône pour tester l'URL. (S'ouvre dans une nouvelle fenêtre ou un nouvel onglet du navigateur.)

Les champs Temps d'arrêt et Reéssayer déterminent combien de temps il faut attendre que le serveur web renvoie la page, et combien de fois il faut réessayer la connexion en cas de défaillance. Si toutes les tentatives sont épuisées, un événement d'exception de source de données est déclenché.
