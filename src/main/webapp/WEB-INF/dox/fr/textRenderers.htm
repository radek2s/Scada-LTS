Vue d'ensemble

Le formatage permet de mettre en forme une valeur brute afin de la rendre plus explicite pour l'utilisateur. Les options de format dépendent du type de la valeur à afficher. Ainsi seuls les formats applicables à la valeur d'un point s'afficheront dans la liste déroulante du champ Type.

Binaire

Le type Binaire permet à l'utilisateur de définir un label pour représenter chacun des deux états binaires(0 et 1). Tous les labels sont autorisés. Vous pouvez aussi définir une couleur pour chaque état. Exemples typiques d'étiquettes off/on, marche/arrêt, ou vrai/faux.

Le format par défaut pour une valeur binaire est Simple. Il affiche la valeur brute, 0 ou 1.

Numérique-

L'option Analogique permet à l'utilisateur de définir un Format qui précisera par exemple le nombre de décimales à afficher, et la présence de séparateurs de millier. (Consultez la documentation "Format nombre"). Un Suffixe peut également être ajouté pour afficher les unités de la valeur. Les suffixes peuvent contenir des entités HTML tel que °(écrit '°') Pour l'affichage des pages . Consultez le site http://www.w3.org/TR/REC-html40/sgml/entities.html

Le type Simple est le format par défaut pour les valeurs numériques. Il affiche la valeur sans modification.

Le type Plage permet de définir un format de nombre Format (Consultez la documentation "Format nombre"), et des textes de votre choix pour représenter une valeur selon des plages de valeurs. On peut assigner à chaque plage une couleur distincte. Le champ format sera appliqué si la valeur ne correspond à aucune plage définie. Par exemple, on peut trier une température "froid", "normal" et "chaud", et associer une couleur pour chaque plage.

Le type Temps permet à l'utilisateur de définir un Format (Consultez la documentation "date/time format"). Un format de date typique serait "JJ,MM,yyyy". Une heure serait "HH:mm:ss". Les formats Date et heures peuvent être combinés si nécessaire. Il est également possible de convertir des temps en millisecondes. Des exposants puissance 10 seront utilisées pour ces conversions. Par exemples pour convertir des secondes entrez le paramètre 3, pour des microsecondes entrez -3. Si la valeur est déjà en millisecondes, entrez 0. Les valeurs de temps sont supposées débuter à 0 le 1 janvier 1970 à minuit

Multistate

L'option Simple est le format par défaut pour les valeurs multistate. Cela affiche la valeur brute sans changement.

l'option Multistate permet d'associer un texte, arbitrairement choisi, à la valeur d'un état pour représenter les différents états du point. Chaque état peut aussi prendre une couleur différente. Si un état n'a pas de label associé, la valeur sera affichée. Exemples typiques de labels on/off/désactivé, ou off/chaud/froid.

Alphanumerique

L'option Simple est le seul format alphanumerique. Les valeurs brutes seront affichées, accompagnées en option d'un Suffixe.
